## 1. Variables and Assignments

A **variable** is a name that refers to a value stored in memory. In Python, you don't need to declare the type — Python figures it out automatically.

```python
x = 10          # x is an integer
name = "Aarav"  # name is a string
pi = 3.14       # pi is a float
```

### Dynamic Typing
Python is **dynamically typed** — the same variable can hold different types at different times.

```python
x = 10      # x is int
x = "hello" # now x is str
x = 3.14    # now x is float
```

### Multiple Assignments
```python
a, b, c = 1, 2, 3       # Multiple variables at once
x = y = z = 0           # Same value to multiple variables
```

## 2. Simple Input and Output

### `input()` — Always Returns a String
```python
name = input("Enter your name: ")
age = input("Enter your age: ")
```
⚠️ **Important:** `input()` **always** returns a string, even if the user types a number.

### `print()` — Displaying Output
```python
print("Hello", name)                    # Multiple arguments, space-separated
print("Age:", age)
print(f"Hello {name}, you are {age}")   # f-string (formatted string)
```

## 3. Data Types

Python's core built-in data types:

| Type | Keyword | Example | Mutable? |
|---|---|---|---|
| Integer | `int` | `10`, `-5`, `0` | ❌ Immutable |
| Float | `float` | `3.14`, `-0.5`, `2.0` | ❌ Immutable |
| String | `str` | `"hello"`, `'world'` | ❌ Immutable |
| Boolean | `bool` | `True`, `False` | ❌ Immutable |
| None | `NoneType` | `None` | ❌ Immutable |
| List | `list` | `[1, 2, 3]` | ✅ Mutable |
| Tuple | `tuple` | `(1, 2, 3)` | ❌ Immutable |
| Dictionary | `dict` | `{"name": "Aarav"}` | ✅ Mutable |

### Mutable vs Immutable
- **Immutable:** Cannot change after creation (`int`, `float`, `str`, `tuple`, `bool`)
- **Mutable:** Can be modified after creation (`list`, `dict`)

## 4. Operators

### Arithmetic Operators

| Operator | Name | Example | Result |
|---|---|---|---|
| `+` | Addition | `5 + 3` | `8` |
| `-` | Subtraction | `10 - 4` | `6` |
| `*` | Multiplication | `6 * 7` | `42` |
| `/` | Division | `10 / 3` | `3.333...` |
| `//` | Floor Division | `10 // 3` | `3` |
| `%` | Modulo | `10 % 3` | `1` |
| `**` | Exponentiation | `2 ** 3` | `8` |

### Comparison (Relational) Operators

| Operator | Name | Example | Result |
|---|---|---|---|
| `==` | Equal to | `5 == 5` | `True` |
| `!=` | Not equal | `5 != 3` | `True` |
| `>` | Greater than | `5 > 3` | `True` |
| `<` | Less than | `5 < 3` | `False` |
| `>=` | Greater or equal | `5 >= 5` | `True` |
| `<=` | Less or equal | `5 <= 3` | `False` |

### Logical Operators

| Operator | Description | Example | Result |
|---|---|---|---|
| `and` | True if both are True | `True and False` | `False` |
| `or` | True if at least one is True | `True or False` | `True` |
| `not` | Reverses the boolean | `not True` | `False` |

### Assignment Operators

| Operator | Example | Equivalent To |
|---|---|---|
| `=` | `x = 5` | Assign 5 to x |
| `+=` | `x += 3` | `x = x + 3` |
| `-=` | `x -= 3` | `x = x - 3` |
| `*=` | `x *= 3` | `x = x * 3` |
| `/=` | `x /= 3` | `x = x / 3` |

## 5. Type Conversion (Type Casting)

### Implicit Conversion (Coercion)
Python automatically converts types when needed:
```python
x = 10      # int
y = 3.5     # float
z = x + y   # z becomes 13.5 (float) — Python upgraded int to float
```

### Explicit Conversion (Type Casting)
You force the conversion:
```python
int("42")       # 42
float("3.14")   # 3.14
str(42)         # "42"
bool(1)         # True
```

⚠️ **Common Trap:**
```python
int("3.14")     # ERROR! Can't convert decimal string directly to int
int(float("3.14"))  # Correct: 3
```

### Truthiness in Python
Every value has a boolean equivalent:

| Value | Boolean |
|---|---|
| `0`, `0.0`, `""`, `[]`, `()`, `{}`, `None` | `False` |
| Any non-zero number, non-empty string, non-empty collection | `True` |
| `bool("False")` | `True` (non-empty string!) |

## 6. Expressions and Operator Precedence

Python evaluates expressions in this order (highest to lowest):

1. `**` (exponentiation)
2. `*`, `/`, `//`, `%` (multiplication, division, floor division, modulo)
3. `+`, `-` (addition, subtraction)
4. `==`, `!=`, `>`, `<`, `>=`, `<=` (comparison)
5. `not` (logical NOT)
6. `and` (logical AND)
7. `or` (logical OR)

```python
result = 5 + 3 * 2 ** 2
# Step 1: 2 ** 2 = 4
# Step 2: 3 * 4 = 12
# Step 3: 5 + 12 = 17
```

## Key Takeaways

1. **Variables are names, not boxes.** They point to values. Python is dynamically typed.
2. **`input()` always returns a string.** Convert explicitly if you need numbers.
3. **`/` gives float, `//` gives int.** Know the difference.
4. **`=` assigns, `==` compares.** Don't mix them up.
5. **Immutable types can't change after creation.** Strings, numbers, tuples — once made, they're fixed.
6. **Every value has a boolean equivalent.** Empty = False, non-empty = True.
