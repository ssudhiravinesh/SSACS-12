## 🏗️ List Basics & Memory
A **list** is a sequence of values (elements). Unlike strings, lists are **mutable** — you can change them in place!

```python
L = [1, 2, 3]
L[0] = 99
print(L)  # [99, 2, 3] -> It changed!
```

---

## 🛠️ Essential List Methods

| Method | What it does | Example (`L = [1, 2]`) |
|---|---|---|
| `append(x)` | Adds `x` to the end | `L.append(3)` -> `[1, 2, 3]` |
| `extend(list)`| Adds all elements from another list | `L.extend([3, 4])` -> `[1, 2, 3, 4]` |
| `insert(i, x)`| Inserts `x` at index `i` | `L.insert(0, 0)` -> `[0, 1, 2]` |
| `pop(i)` | Removes and returns item at index `i` | `L.pop(0)` -> returns `1`, `L` is `[2]` |
| `remove(x)` | Removes the first occurrence of value `x` | `L.remove(1)` -> `L` is `[2]` |
| `sort()` | Sorts the list (in-place) | `[3, 1].sort()` -> `[1, 3]` |
| `reverse()` | Reverses the list (in-place) | `[1, 2].reverse()` -> `[2, 1]` |

---

## 🪜 Nested Lists (Lists within Lists)
Think of a nested list as a **matrix** or a table.

```python
matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
]

# Accessing elements: matrix[row][column]
print(matrix[0][1]) # Output: 2
print(matrix[2][2]) # Output: 9
```

---

## ⚠️ The "Alias Trap"
If you set `L2 = L1`, they both point to the **same** list in memory. Changing one changes both!
To make a real copy, use slicing: `L2 = L1[:]`.

## 💡 Key Takeaway
Lists are powerful because they can be modified. However, this means you have to be careful about **aliasing** (two variables pointing to the same list). When in doubt, trace it out!
