## 🏗️ String Basics & Memory
A **string** is a sequence of characters. In Python, they are **immutable** — this means once created, they cannot be changed in place.

```python
s = "Hello"
# s[0] = "M"  <-- This will cause a TypeError!
s = "Mello"    # This creates a NEW string object in memory.
```

---

## ✂️ The Power of Slicing
Slicing allows you to extract a "slice" of a string.
Syntax: `string[start:stop:step]`

- `start`: Inclusive (where to begin)
- `stop`: Exclusive (where to end — Python stops *before* this index)
- `step`: How many characters to skip (default is 1)

**Cool Tricks:**
- `s[::-1]` — Reverses the entire string.
- `s[::2]` — Gets every second character.
- `s[5:]` — From index 5 to the end.

## 🛠️ Essential String Methods
These methods return a **new string** (remember, strings are immutable!).

| Method | What it does | Example |
|---|---|---|
| `upper()` | Converts to ALL CAPS | `"hello".upper() -> "HELLO"` |
| `lower()` | Converts to lowercase | `"HELLO".lower() -> "hello"` |
| `capitalize()`| Capitalizes first letter | `"hi there".capitalize() -> "Hi there"` |
| `strip()` | Removes leading/trailing whitespace | `"  hey  ".strip() -> "hey"` |
| `find(sub)` | Finds index of substring | `"python".find("th") -> 2` |
| `replace(a, b)`| Replaces all `a` with `b` | `"apple".replace("p", "b") -> "abble"` |
| `isalnum()` | Returns True if all chars are letters/numbers | `"abc12".isalnum() -> True` |
