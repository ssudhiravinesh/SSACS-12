## 1. Tokens in Python

The smallest individual unit in a Python program is called a **token** (or lexical unit).

### Five Types of Tokens

| Type | Description | Examples |
|---|---|---|
| **Keywords** | Reserved words with special meaning | `if`, `else`, `for`, `while`, `def`, `class`, `import`, `True`, `False`, `None` |
| **Identifiers** | User-defined names for variables, functions, etc. | `total`, `_count`, `myVar`, `student_name` |
| **Literals** | Constant data items that never change | `42`, `3.14`, `"hello"`, `True`, `None` |
| **Operators** | Symbols/words that perform operations | `+`, `-`, `*`, `/`, `==`, `and`, `not` |
| **Punctuators** | Symbols that structure the code | `(`, `)`, `[`, `]`, `{`, `}`, `:`, `,` |

### Rules for Identifiers
- Must start with a **letter** (a-z, A-Z) or **underscore** (_)
- Can contain letters, digits, and underscores
- Cannot be a keyword
- Case-sensitive (`Name` ≠ `name`)
- Cannot contain special characters like `-`, `*`, `@`, `$`

**Valid:** `_total`, `name2`, `student_name`, `MyVar`  
**Invalid:** `2ndPlace` (starts with digit), `my-var` (hyphen), `class` (keyword), `Price*Qty` (operator)

## 2. Barebones of a Python Program

A minimal Python program consists of:
1. **Comments** (optional) — `#` for single line, `''' '''` for multi-line
2. **Import statements** (optional) — `import math`
3. **Variable assignments** — `x = 10`
4. **Statements** — instructions executed by Python
5. **Indentation** — defines code blocks (not braces)

```python
# This is a comment
import math

radius = 5
area = math.pi * radius ** 2
print(f"Area: {area:.2f}")
```

## 3. Mutable vs Immutable Types

| Mutable (can change) | Immutable (cannot change) |
|---|---|
| `list` | `int`, `float`, `str` |
| `dict` | `tuple` |
| `set` | `bool`, `NoneType` |

### Proof:

**Immutable (string):**
```python
s = "hello"
# s[0] = "H"  # TypeError! Strings can't be modified in-place
s = "Hello"   # Works — creates a NEW string, reassigns s
```

**Mutable (list):**
```python
L = [1, 2, 3]
L[0] = 99     # Works! Lists can be modified in-place
print(L)      # [99, 2, 3]
```

## 4. Math Library Functions

```python
import math
```

| Function | Description | Example | Result |
|---|---|---|---|
| `math.sqrt(x)` | Square root | `math.sqrt(25)` | `5.0` |
| `math.ceil(x)` | Round up | `math.ceil(3.2)` | `4` |
| `math.floor(x)` | Round down | `math.floor(3.8)` | `3` |
| `math.fabs(x)` | Absolute value (float) | `math.fabs(-7.5)` | `7.5` |
| `math.pow(x, y)` | x raised to y | `math.pow(2, 3)` | `8.0` |
| `math.exp(x)` | e^x | `math.exp(1)` | `2.718...` |
| `math.log(x)` | Natural log (ln) | `math.log(math.e)` | `1.0` |
| `math.log10(x)` | Base-10 log | `math.log10(100)` | `2.0` |
| `math.sin(x)` | Sine (radians) | `math.sin(0)` | `0.0` |
| `math.cos(x)` | Cosine (radians) | `math.cos(0)` | `1.0` |
| `math.pi` | π constant | `math.pi` | `3.14159...` |
| `math.e` | e constant | `math.e` | `2.71828...` |

⚠️ `math.sqrt()`, `math.pow()` always return **float**.  
⚠️ `math.log()` is undefined for `x ≤ 0` — will raise `ValueError`.

## 5. Statement Flow Control

Python executes statements in three ways:

| Flow Type | Description | Constructs |
|---|---|---|
| **Sequential** | Top to bottom, one after another | Default behavior |
| **Conditional** | Execute based on a condition | `if`, `if-else`, `if-elif-else` |
| **Iterative** | Repeat a block of code | `for`, `while` |

## 6. The `if` Conditionals

### Plain `if`
```python
num = int(input("Enter a number: "))
if num > 0:
    print("Positive")
```

### `if-else`
```python
if num % 2 == 0:
    print("Even")
else:
    print("Odd")
```

### `if-elif-else`
```python
marks = int(input("Enter marks: "))

if marks >= 90:
    grade = "A+"
elif marks >= 80:
    grade = "A"
elif marks >= 70:
    grade = "B"
elif marks >= 60:
    grade = "C"
elif marks >= 50:
    grade = "D"
else:
    grade = "F"
```

### Nested `if`
```python
year = int(input("Enter year: "))

if year % 4 == 0:
    if year % 100 == 0:
        if year % 400 == 0:
            print("Leap year")
        else:
            print("Not a leap year")
    else:
        print("Leap year")
else:
    print("Not a leap year")
```

### Storing Conditions
```python
is_adult = age >= 18  # Stores True or False
if is_adult:
    print("You can vote")
```

## Key Differences

### `elif` vs Multiple `if`
```python
# Using elif — stops after first match
if x > 10:
    print("A")
elif x > 5:   # Only checked if first condition is False
    print("B")

# Using separate if — ALL conditions checked
if x > 10:
    print("A")
if x > 5:     # Always checked, even if first was True
    print("B")
```

### `=` vs `==`
- `=` is **assignment**: `x = 5` (stores 5 in x)
- `==` is **comparison**: `x == 5` (checks if x equals 5, returns True/False)

## Key Takeaways

1. **Tokens are the building blocks** of every Python program. Keywords are reserved — you can't use them as variable names.
2. **Mutable types can change in-place.** Immutable types create new objects when "changed."
3. **`math` module provides essential functions.** Always `import math` first.
4. **`elif` is not just style — it's efficiency.** It skips remaining checks once one matches.
5. **Indentation defines blocks in Python.** It's not optional — it's syntax.
6. **Nested `if` = `if` inside `if`.** The `else` matches the nearest unmatched `if`.
