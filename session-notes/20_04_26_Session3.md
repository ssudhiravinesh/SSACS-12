## 1. Why do we need Loops?
In programming, we often need to repeat a block of code multiple times. Instead of writing the same line 100 times, we use **Loops**.

There are two main types of loops in Python:
1. **`for` loop:** Used when you know (or can calculate) how many times to repeat.
2. **`while` loop:** Used when you want to repeat as long as a condition is True.

---

## 2. The `for` loop and `range()`

The `for` loop is usually used with the `range()` function.

### Syntax of `range()`:
- `range(stop)`: From 0 to stop-1.
- `range(start, stop)`: From start to stop-1.
- `range(start, stop, step)`: From start to stop-1, jumping by 'step'.

### Example:
```python
# Print 0 to 4
for i in range(5):
    print(i)

# Print 1 to 5
for i in range(1, 6):
    print(i)

# Print even numbers from 2 to 10
for i in range(2, 11, 2):
    print(i)
```

**⚠️ Important:** Python always stops one step BEFORE the 'stop' value.

---

## 3. The `while` loop

A `while` loop repeats as long as a condition remains `True`.

### Example:
```python
count = 1
while count <= 5:
    print(count)
    count += 1  # Don't forget this, or the loop will never end!
```

---

## 4. Jump Statements: `break` and `continue`

Sometimes we need to change the flow of a loop from the inside.

- **`break`:** Exits the loop completely.
- **`continue`:** Skips the rest of the CURRENT turn and goes to the start of the next turn.

### Example:
```python
for i in range(1, 10):
    if i == 5:
        break  # Loop stops completely when i is 5
    print(i)

for i in range(1, 6):
    if i == 3:
        continue  # Skips 3, but prints 1, 2, 4, 5
    print(i)
```

---

## 5. The Loop `else` Clause

Python has a unique feature where you can add an `else` to a loop.
- The `else` block runs **only if the loop finishes normally** (without hitting a `break`).
- If a `break` is triggered, the `else` is skipped.

---

## 6. Nested Loops

A loop inside another loop. The inner loop runs completely for every single turn of the outer loop.

### Example:
```python
for i in range(3):        # Outer loop
    for j in range(2):    # Inner loop
        print(f"i={i}, j={j}")
```

## Key Takeaways
- Use `for` for fixed counts; `while` for conditions.
- `range(start, stop)` excludes `stop`.
- `break` = Exit; `continue` = Skip current turn.
- `else` in a loop = "If the loop wasn't broken."
