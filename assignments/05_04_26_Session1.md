## Tier 2: Application

### Q4. Smart Calculator
Write a program that:
1. Takes two numbers from the user
2. Asks which operation: `+`, `-`, `*`, `/`, `//`, `%`, `**`
3. Performs the operation and prints the result formatted
4. If the user enters an invalid operation, print an error message
5. If the user chooses division and enters 0 as the second number, handle it gracefully (don't crash)

**Example:**
```
Enter first number: 10
Enter second number: 3
Choose operation (+, -, *, /, //, %, **): %

Result: 10 % 3 = 1
```

**Example:**
```

After:  a = 2, b = 3, c = 1
```

Do it in **ONE line**. No temporary variables.
